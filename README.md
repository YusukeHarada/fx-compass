# FX-Compass Pro (為替羅針盤 Pro)

FX-Compass Proは、MACD、RSI、および価格動向をエンジニアリング視点で複合的に分析し、複数の通貨ペアから「真のチャンス」を抽出するための監視ツールです。

## 1. コア・コンセプト：3段階スコアリング

単なる売買サインではなく、条件の重なり具合に応じた3段階の「信頼度」を定義しています。

| レベル | 判定名称 | 条件・ロジック | 視覚表示 (チャート上) |
| :--- | :--- | :--- | :--- |
| **Lv.1** | **WATCH** | MACDのゴールデンクロス/デッドクロスが発生 | 通常サイズの矢印 (緑/赤) |
| **Lv.2** | **STANDARD** | Lv.1 + MACDが0ライン以下(買) / 以上(売) | 中サイズの矢印 (緑/赤) |
| **Lv.3** | **STRONG** | Lv.2 + RSIが30-50(買) / 50-70(売) | **金色の特大矢印** |

## 2. 主な機能

- **マルチシンボル・スキャン**: `config.yaml` にリストアップした全通貨ペアを自動で巡回分析。
- **ヒストリカル・シグナル表示**: 「今」だけでなく、チャート上の過去100本分に含まれる全シグナルを可視化。
- **動的停止線 (Stop Loss)**: 判定時の価格からリスク許容度に応じた損切りラインを自動算出。
- **スマート・データ取得**: 1分足などの短期足選択時、API制限を回避するために取得期間を自動調整（1日分に限定）。

## 3. クイップスタート

### 必要ライブラリのインストール
```bash
pip install yfinance pandas pandas_ta PyYAML matplotlib
```

### 設定ファイル (config.yaml) の編集
監視したい通貨ペアをリスト形式で自由に追加できます。
```yaml
trading:
  symbols:
    - "USDJPY=X"
    - "AUDJPY=X"
    - "MXNJPY=X"
    - "TRYJPY=X"
  interval: "1h"
  period: "3mo"
```

### 実行
```bash
python main.py
```

## 4. 分析チャートの見方

生成されるチャートは以下の3段構成になっています。

1.  **Main Chart (上段)**: 価格推移と売買シグナルの矢印。レベルに応じてサイズと色が変化します。
2.  **MACD (中段)**: トレンドの転換点と0ラインとの位置関係。シグナルの信頼度を測る指標です。
3.  **RSI (下段)**: 相場の過熱感。Lv.3判定に必須の「反発余地」を確認します。

## 5. 推奨される監視通貨ペア例

- **主要通貨**: `USDJPY=X`, `EURUSD=X`, `GBPJPY=X`
- **高金利・資源国通貨**: `AUDJPY=X`, `MXNJPY=X`, `TRYJPY=X`
- **仮想通貨 (参考)**: `BTC-USD`, `ETH-USD`
